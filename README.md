# BancoDePalavras
Portfólio - Aplicativo banco de palavras.


Aplicativo que obtém palavras que serão usadas para brincadeira de mímica de maneira que cada palavra possui um nível:

- Nível fácil.
- Nível médio.
- Nível difícil.

Requisitos:

- Webservice.
- Cadastro de palavras e seus níveis.
- Autenticação.
